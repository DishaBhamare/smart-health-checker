# 🩺 Smart Health Checker

Smart Health Checker is a full-stack web application that allows users to input symptoms and get possible disease predictions along with confidence score, severity level, and risk assessment.

---

## 🚀 Features

- 🔐 User Authentication (Register & Login)
- 🧠 Symptom-based disease prediction
- 📊 Top 3 possible diagnoses
- 📈 Confidence score calculation
- ⚠️ Risk Level (Low / Moderate / High)
- 📁 User history tracking
- 👤 Profile management
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

**Frontend:**
- HTML, CSS, JavaScript
- Bootstrap

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

---

## ⚙️ How It Works

1. User selects symptoms
2. Backend processes using:
   - Database-based weighted scoring
   - Rule-based fallback system
3. Returns:
   - Top matching diseases
   - Confidence %
   - Severity & Risk Level
